This folder contains the code that will be use to give the Robot intelligence. 

The AIML module was downloaded from:
https://github.com/creatorrr/pyAIML

This module is used to parse the inputs into usable information that is interpreted using the AIML files.

The AIML library of files was downloaded from:
https://code.google.com/archive/p/aiml-en-us-foundation-alice/
This is a free set of AIML files that were created and updated for people to use on their own projects.
